# AnchorChain v2 Site

A Vercel-ready Next.js landing page for AnchorChain.

## Logo placement

Put your chosen logo file here:

`public/anchorchain-logo.jpg`

If you want to use PNG or SVG instead, update the filename in:

`app/page.tsx`

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to a GitHub repo.
2. Go to Vercel and import that repo.
3. Vercel will auto-detect Next.js.
4. Click Deploy.
